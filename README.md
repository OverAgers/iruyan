## 環境構築

### 1. Docker イメージのビルドとコンテナの起動

以下のコマンドで、Docker コンテナをビルドし、バックグラウンドで起動します。

```bash
docker-compose up -d --build
```

### 2. コンテナの終了

コンテナを停止・削除するためには、以下のコマンドを使用します。

```bash
docker-compose down
```
